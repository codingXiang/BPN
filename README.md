# 倒傳遞神經網路 演算法實作
利用 Python 實作 Machine Learning 演算法 - 倒傳遞神經網路（BPN）

## 資料集
#### 使用 UCI 的 Iris（鳶尾花）資料集

## 參數說明
#### 網路架構
- 輸入層（x）
使用 Iris 的四個特徵值作為輸入層，為一個（150 * 3）的矩陣
- 隱藏層（h）
設定為一個（150 * 3）的矩陣
- 輸出層（Y）
使用 Iris 的輸出為三類，為一個（150 * 4）的矩陣
